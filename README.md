# Install
```
mkdir ~/workspace
cd ~/workspace

git clone https://github.com/fleth/stock_setup.git
git clone https://github.com/fleth/stock_chart.git
git clone https://github.com/fleth/stock_data_sample.git stock_data

cd stock_setup
sh install.sh
```

# Usage
```
python run.py
```
